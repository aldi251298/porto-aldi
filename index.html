Building Scalable Fullstack Applications

Web Developer yang berfokus pada pembangunan aplikasi web performa tinggi menggunakan framework modern dan arsitektur database yang terstruktur.

Eksplorasi Teknologi & Pengembangan

Saya adalah seorang pengembang web yang antusias dalam membangun aplikasi dari tahap perancangan hingga deployment. Dengan spesialisasi pada ekosistem TypeScript dan framework modern seperti SvelteKit, saya berfokus pada penciptaan kode yang bersih, mudah dipelihara (*maintainable*), dan siap untuk lingkungan produksi.

Pengalaman saya mencakup pengelolaan database PostgreSQL menggunakan Prisma ORM, optimasi performa melalui Redis caching, hingga integrasi frontend yang responsif dengan Tailwind CSS. Saya percaya bahwa pengembangan web yang baik adalah perpaduan antara logika backend yang kuat dan pengalaman pengguna yang mulus.
